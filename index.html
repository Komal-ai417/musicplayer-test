My C++ Music Player Project

A simple, command-line based music playlist manager.

About The Project

This is a command-line application written in C++ that allows users to manage a playlist of songs. It uses a doubly-linked list to store the songs and supports features like adding, removing, searching, and saving playlists to a file. It also includes shuffle and repeat modes.

How to Compile and Run

To run this project on your own machine:

  1. Clone the repository from GitHub.
  2. Make sure you have a C++ compiler (like g++).
  3. Navigate to the project directory and compile the code using the command:
  4. g++ -std=c++17 -o music_player main.cpp
  5. Run the executable: 
    ./music_player
